# 📌 Quadro de Tarefas

Aplicação de gerenciamento de tarefas inspirada em sistemas Kanban, desenvolvida com HTML, CSS e JavaScript puro.

## 🚀 Funcionalidades

- ✅ Adicionar tarefas
- ✏️ Editar tarefas
- 🗑️ Remover tarefas
- 📂 Separação por status:
  - Pendente
  - Em andamento
  - Concluído
- 💾 Salvamento automático com localStorage
- 🔢 Contador automático de tarefas
- ⌨️ Adicionar tarefa pressionando Enter
- 🎨 Interface responsiva e moderna

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📚 Conceitos praticados

- Manipulação do DOM
- Eventos
- Arrays e objetos
- Métodos como:
  - `forEach`
  - `filter`
  - `splice`
- Organização de código
- Renderização dinâmica
- Persistência de dados com localStorage
